Tipos de cliente:

Avançado oferece um conjunto completo de recursos de colaboração na Web. Este Cliente para Web funciona melhor com navegadores mais novos e conexões de Internet mais rápidas.

Padrão é recomendado quando as conexões de Internet forem lentas, quando usar navegadores antigos ou para melhor acessibilidade.

Móvel é recomendado para dispositivos móveis.

Tablet é recomendado para tablets.

Para definir Padrão como seu tipo de cliente preferido, altere as opções de login em suas Preferências, aba Geral após efetuar o login.
Observe que seu navegador ou monitor não é totalmente compatível com a versão avançada. Recomendamos usar o cliente padrão.
